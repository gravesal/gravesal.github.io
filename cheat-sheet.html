SQL Cheat Sheet

What is SQL?

SQL (Structured Query Language) is a special-purpose programming language designed for managing data held in a relational database management system (RDBMS), or for stream processing in a relational data stream management system (RDSMS)

-Wikipedia

Tips for Success in Topic

  1. Always use a semicolon after an argument.
  2. Make sure data types are corect.
  3. "Inner Joins" return matching records.
  4. "Outer joins" do not require each record to have a matching record.
  5. A left outer join of tables A and B will return all matching records of A and B, as well as any non-matched records from the left table, in this case, A.
  6. A right outer join of tables A and B will return all matching records of A and B, as well as any non-matched records from the right table, in this case, B.
  7. A full outer join of tables A and B will return all matching records of A and B, as well as any non-matched records from both tables.
  8. A self join is merely a join from a table to itself.

Useful Commands

SELECT Extracts data from a database
UPDATE Updates data in a database
DELETE Deletes data from a database
INSERT INTO Inserts new data into a database

CREATE TABLE creates a new table
DROP TABLE deletes a table
